# MEMORY.md — 跨会话项目记忆

> **AI 每次开工前必须读取此文件。**
>
> 此文件由系统自动维护（`update_memory.py` hook 在每次 `/opsx:archive` 后追加）。
> 也可以人工追加，格式见底部模板区。
>
> 内容优先级高于 `implicit-contracts.md`（后者为架构约定，此文件为实战经验）。

---

## 如何阅读本文件

每条记录对应一次已归档的 OpenSpec 变更，包含：
- **变更摘要**：做了什么
- **关键决策**：为什么这么设计（方便后续变更不重蹈覆辙）
- **踩坑记录**：实现过程中遇到的问题和解法
